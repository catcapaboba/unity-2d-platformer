# Unity 2D Platformer

Educational mini-project created in Unity.

## About the project
This project is a simple 2D platformer prototype made for practice in Unity and C#.

## Features
- character movement
- jumping
- basic collisions
- Rigidbody2D physics
- UI elements
- start/restart functionality

## Technologies
- Unity
- C#

## Purpose
The goal of this project was to practice basic game mechanics, physics and UI implementation in Unity.
